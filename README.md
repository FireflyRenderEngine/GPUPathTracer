# GPUPathTracer
A sophisticated GPU Path Tracer.

## Prerequisites:
- CUDA 10.1

## Progress
 - GLFW/GLAD Viewer
   - JSON scene loading (See [scenes/testscene.json](scenes/testscene.json))
 - OBJ Files support
 - Native geometry support
   - Sphere
   - Plane
   - Cube
   - Triangle Mesh
